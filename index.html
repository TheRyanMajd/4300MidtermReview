HTML

  1. Relative and absolute paths

    Relative Path: Specifies the path relative to the current directory. Absolute Path: Specifies the complete path including the protocol and domain.

  2. Div tags

    The <div> tag is used to group block-level content and is often used for styling or as a container for other elements.

  3. Required HTML elements - boilerplate - html, head, title, body

    Essential HTML boilerplate elements include <html>, <head>, <title>, and <body>.

  4. Block vs inline elements

    Block elements start on a new line and take full width. Inline elements do not start on a new line and take only necessary width.

  5. Create link, img, headings, list

    Links are created with <a>, images with <img>, headings with <h1>-<h6>, and lists with <ul>/<ol> and <li>.

  6. Attributes of img, a elements

    <img> attributes include src, alt, width, height. <a> attributes include href, target, rel.

  7. Font units/families - fixed vs dynamic

    Fixed units like px do not scale, whereas dynamic units like em, rem, and % scale with parent or root element's font size.

  1. Forms - collect data

    Forms collect user input through elements like <input>, <textarea>, <button>, <select>, and <option>.



CSS

Internal vs External CSS

  • Internal CSS is written inside the HTML using a <style> element.
  • External CSS is stored in a separate file (e.g., style.css) and linked to the HTML.

CSS Selectors

  • Element Selector: Targets HTML elements directly.
  • Class Selector: Targets elements with a specific class attribute. Written as .classname.
  • ID Selector: Targets an element with a specific id attribute. Written as #id.

Descendant Selectors

Descendant selectors in CSS are used to select an element that is nested within another element, regardless of the depth of nesting. Written with a space between two or more selectors. Styles are applied to the innermost selector if it is a descendant of the outer selector(s).

Other CSS Concepts

  • Pseudo-classes (e.g., :hover) are used to define the special state of an element.
  • The CSS Box Model describes the design and layout of elements, including margins, borders, padding, and the actual content.
  • Responsive web design ensures web applications look good on all devices.
  • Responsive layouts can be achieved using float, flexbox, or CSS grid.
  • Grid layouts establish a column structure for placing elements.
  • Flexbox is used for spacing layout and aligning items in a container.
  • Media queries allow the application of CSS rules based on device characteristics, such as screen width.
  • Centering block elements can be done using various CSS techniques.

JavaScript

Run-time Environment

JavaScript can run in a browser or on a server (using Node.js).

Functions

Callbacks & Higher Order Functions

A higher-order function is a function that takes another function as an argument or returns a function.

function higherOrderFunction(callback) { callback(); }

Anonymous and Arrow Functions

const arrowFunction = () => console.log('This is an arrow function');

Variables & Scope

JavaScript supports const, let, and var for declaring variables.

JavaScript Types

JavaScript provides types like Number, String, Boolean, Object, etc.

DOM Manipulation

The Document Object Model (DOM) can be manipulated to change the document structure, style, and content.

document.querySelector('h1').textContent = 'Changed Heading';

Selecting Nodes

const elementById = document.getElementById('id'); const elementsByClass = document.getElementsByClassName('class'); const elementsByTag = document.getElementsByTagName('tag');

Asynchronous JavaScript

Callbacks

function fetchData(callback) { // Simulate fetching data setTimeout(() => callback('Data'), 1000); }

Promises

const promise = new Promise((resolve, reject) => { // Simulate async operation setTimeout(() => resolve('Data'), 1000); });

Async/Await

async function fetchDataAsync() { const data = await promise; console.log(data); }

Event Handling

document.getElementById('button').addEventListener('click', () => { console.log('Button clicked'); });

Array Methods

Common methods include forEach, map, filter, sort, and reduce.

Example: Arrow Function

const add = (a, b) => a + b;

Example: Higher Order Function

function repeat(n, action) { for (let i = 0; i < n; i++) { action(i); } } repeat(3, console.log); // Logs 0, 1, 2

Example: DOM Event Listener

document.querySelector('button').addEventListener('click', () => { console.log('Button was clicked'); });

Example: Handling Asynchronous Calls

function getData() { return new Promise(resolve => { setTimeout(() => resolve('Some data'), 1000); }); } getData().then(data => console.log(data));
